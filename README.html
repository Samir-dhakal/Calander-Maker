Java Calendar Maker

This is a simple Java console application that takes a year and month as input and prints the calendar for that month.

Features

  • Prompts user input for year and month
  • Calculates the first day of the month
  • Handles leap years for February
  • Displays the calendar in a nicely formatted grid

How to Use

  1. Clone the repository: 
    git clone https://github.com/YOUR-USERNAME/YOUR-REPO.git
  2. Navigate to the project directory: 
    cd YOUR-REPO
  3. Compile the Java program: 
    javac CalenderMaker.java
  4. Run the program: 
    java CalenderMaker
  5. Follow the prompts to enter a year and month.

Example

enter year 
2025
enter month
5

  • CalenderMaker.java: Main Java program that generates the calendar.
  • README.html: Project description and usage instructions.

Notes

  • The program uses Zeller's Congruence algorithm to determine the first day of the month.
  • Leap year calculation is included.
